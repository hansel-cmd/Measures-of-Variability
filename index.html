Measures of Variability

Range:

Formula: Range = Highest value - Lowest value

Shows how separate (or the distance) the largest data value from the smallest data value

Mean:

Formula: μ = ∑ given_number / total_number

μ is the mean

Population Variance:

Formula: σ² = ∑(X - μ)² / N

Where N is the total number of data

Where X is each data

Where μ is the population mean

Is the average of the squares of the distance each value is from the population mean

Population Standard Deviation:

Formula: σ = √σ²

It refers to how far the value is to the mean

Is the square root of the variance

Sample Variance:

Formula: S² = ∑(X - X̄)² / (n - 1)

Where n is the total number of data

Where X is each data

Where X̄ is the sample mean

Is the average of the squares of the distance each value is from the sample mean

Sample Standard Deviation:

Formula: S = √S²

It refers to how far the value is to the mean

Is the square root of the variance
